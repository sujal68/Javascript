Q.1 What is JavaScript?

JavaScript is a programming language used to make websites interactive, dynamic, and responsive.

Q.2 What are JavaScript logical operators?

Logical operators are used to combine or invert boolean values: AND (&&), OR (||), NOT (!)

Q.3 What is a ternary operator?

A short if-else statement: condition ? valueIfTrue : valueIfFalse

Q.4 What is hoisting?

Hoisting is JavaScript's behavior of moving variable and function declarations to the top of their scope.

Q.5 Define for, while, and do-while loops

for: Loop with initialization, condition, increment
while: Loop while condition is true
do-while: Executes at least once, then checks condition

Q.6 What is a closure? Give an example

Closure is a function that remembers the variables of its outer function even after the outer function has finished execution. Example: function outer(){ let x=10; return function(){ console.log(x); } }

Q.7 Difference between let, var, and const

var: function-scoped, can be redeclared
let: block-scoped, cannot be redeclared
const: block-scoped, cannot be redeclared or reassigned

Q.8 Difference between function and arrow function

function: has its own this
arrow function: inherits this from parent scope

Q.9 Difference between null and undefined

null: explicitly assigned empty value
undefined: variable declared but not assigned

Q.10 What is the spread operator?

... allows expanding arrays or objects. Example: let arr2=[...arr1];

Q.11 Difference between callback and callback hell

Callback: function passed as argument. Callback Hell: nested callbacks making code messy.

Q.12 Use of charAt(), substring(), includes()

charAt(index): returns character at index
substring(start,end): returns substring
includes(text): returns true if string contains text

Q.13 What are string methods?

Methods to manipulate strings, e.g. toUpperCase(), toLowerCase(), slice(), split(), replace()

Q.14 What is a palindrome?

A string or number that reads the same forwards and backwards. Example: "121" or "madam"

Q.15 What is an array? Name 10 array methods

Array is a collection of elements. Methods: push, pop, shift, unshift, slice, splice, map, filter, reduce, includes

Q.16 Difference between slice() and splice()

slice(start,end): returns new array, does not modify original
splice(start,count,newItems): modifies original array, can add/remove items

Q.17 Remove duplicate elements from an array

Use Set or loops. Example: let unique = [...new Set(arr)];

Q.18 Find first/last/highest number in an array

First: arr[0], Last: arr[arr.length-1], Highest: Math.max(...arr)

Q.19 Find index of a number in an array

arr.indexOf(number)

Q.20 Print array elements into two arrays alternately

Example: arr=[1,2,3,4]; arr1=[1,3]; arr2=[2,4]

Q.21 What is currying in JavaScript?

Currying is transforming a function with multiple arguments into a series of functions with single argument each. Example: f(a)(b) instead of f(a,b)

Q.22 What are Promises?

Promises handle asynchronous operations, they can be pending, resolved, rejected

Q.23 Difference between Promise and async/await

Promise: handles async tasks with .then() async/await: syntax sugar for Promises, makes code look synchronous

Q.24 Synchronous vs Asynchronous in JS

Synchronous: executes line by line, blocking Asynchronous: executes tasks without blocking, like AJAX, setTimeout

Q.25 What is ES6? List some new features

ES6 = ECMAScript 2015 Features: let, const, arrow functions, template literals, default parameters, spread/rest, promises, classes

Q.26 What is a Higher Order Function (HOC)?

A function that takes another function as argument or returns a function. Example: arr.map(func)
