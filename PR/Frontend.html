Q.1 Explain the difference between a block-level and inline element, and how can you force one to behave like the other?

Block-level elements: Ye full width lete hain aur nayi line se start hote hain. Examples: <div>, <p>, <h1>.
Inline elements: Ye sirf jitni width chahiye utni lete hain aur same line me hi rehte hain. Examples: <span>, <a>, <strong>.
Force one to behave like the other: CSS ka display property use karo. For example, display: inline; ya display: block;.

Q.2 What are meta tags and why are they important for websites?

Meta tags page ke bare me browser aur search engines ko info dete hain. Examples: <meta charset="UTF-8">, <meta name="viewport" content="width=device-width, initial-scale=1.0">. Ye SEO, responsiveness, aur proper page rendering ke liye important hain.

Q.3 What are void elements? Can you give a few examples?

Void elements wo HTML elements hain jinke closing tag nahi hote. Examples: <br>, <hr>, <img>, <input>.

Q.4 What is the box model in CSS? Can you draw and explain it?

Box model har element ke layout ko 4 layers me describe karta hai: Content → Padding → Border → Margin. Content: actual text/image, Padding: border ke andar space, Border: element ka edge, Margin: element ke bahar space.

Q.5 What are the differences between HTML4 and HTML5?

- HTML5 me semantic elements support hote hain like <header>, <footer>, <section>. - Audio, video, canvas, localStorage, aur offline capabilities available hain. - HTML4 me ye modern features nahi hote. - HTML5 mobile aur responsive design ke liye better hai.

Q.6 How does the target="_blank" attribute in an anchor tag work? What are its security implications?

target="_blank" link ko new tab me open karta hai. Security risk: new page window.opener access kar sakta hai aur original page ko redirect kar sakta hai. Fix: rel="noopener noreferrer" use karo isse avoid karne ke liye.

Q.7 What is Flex and Grid in CSS?

Flex: 1D layout (row ya column), items ko horizontal ya vertical align karta hai. Grid: 2D layout (rows + columns), complex layouts ke liye use hota hai.

Q.8 How many ways can you add CSS to a project?

1. Inline CSS: <div style="color:red"> 2. Internal CSS: <style> tag use karke 3. External CSS: <link rel="stylesheet" href="style.css">

Q.9 What is the Bootstrap grid system, and how does it work?

Bootstrap grid system ek 12-column responsive layout hai. Content ko <div class="row"> me wrap karo aur columns jaise <div class="col-6"> 50% width ke liye use karo. Columns automatically different screen sizes ke liye adjust ho jate hain.

Q.10 How to write display: flex using Bootstrap?

Container element pe class="d-flex" use karo. Example: <div class="d-flex">...</div>
